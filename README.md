# AI Cancer Detection System

A professional web-based AI system for cancer detection using medical images.  
The project integrates frontend (React.js), backend (Node.js), and AI backend (Python with deep learning models) for automated analysis and reporting.

---

## Features

- Upload medical images for AI-based cancer detection  
- View detection results with precision, recall, and F1 metrics  
- User-friendly frontend dashboard with detailed reports  
- Modular architecture for easy extension and maintenance

---

## Folder Structure

- `frontend/` → React.js web application  
- `backend/` → Node.js server and API routes  
- `ulms-backend/` → Python AI models for detection  

---

## Technologies Used

- **Frontend:** React.js, HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **AI/ML:** Python, PyTorch, YOLO, OpenCV
